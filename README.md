矩阵分析与应用

本项目为矩阵分析与计算（X2MS1012）课程设计成果：基于KSVD和SVD的图像修复与音频降噪综合应用平台。系统集成了KSVD字典学习算法进行图像划痕/污渍修复，以及SVD分帧处理技术进行音频降噪，并提供完整的可视化评估指标。

项目亮点

* KSVD图像修复：支持手动绘制划痕和自动生成污渍两种修复模式
  基于KSVD字典学习算法，动态稀疏度衰减
  边缘精修算法，保证修复区域与原图无缝融合
  支持彩色图像修复，保留原始图像质量
* SVD音频降噪：基于分帧SVD的增强型降噪系统
  混合阈值降噪（软阈值+硬截断）
  自适应截断寻优，自动选择最优k值
  完整的残差分析（波形、频谱、直方图）
* SVD图像去噪评估：矩阵分析与图像处理综合评估
  支持高斯噪声、椒盐噪声、像素丢失等多种噪声类型
  PSNR（峰值信噪比）和SSIM（结构相似性）双指标评估
  SVD与中值滤波算法对比

快速开始（3步）

1. Win+R、 cmd打开命令提示符并进入项目目录
2. 安装依赖：
   pip install -r requirements.txt
3. 启动演示界面：
   streamlit run app.py

打开浏览器，访问本地Streamlit页面（通常为 http://localhost:8501），即可开始体验！

功能模块详解

1. KSVD图像修复

* 划痕修复：手动绘制划痕区域，系统通过邻域填充和KSVD字典学习进行修复
* 污渍修复：自动生成随机污渍（圆形、椭圆形、不规则形状），支持区域深度修复
* 质量评估：PSNR、SSIM、MAE、MSE等多维度指标

2. SVD音频降噪

* 默认测试信号：440Hz+880Hz+1320Hz三音测试信号，包含起振和衰减包络
* 上传WAV文件：支持用户上传自己的音频文件进行处理
* 音乐模式专用：针对音乐音频优化的降噪参数
* 可视化分析：波形对比、奇异值分布、SNR vs k曲线、残差分析

3. SVD图像去噪评估

* 高斯噪声：使用改进的SVD矩阵分解（低秩近似）
* 椒盐噪声/像素丢失：使用中值滤波（空间域处理）- 仅作为对比参考
* 实时评估：PSNR和SSIM指标动态显示

硬件要求

* CPU模式：支持任何现代处理器，处理时间随图像/音频大小线性增加
* GPU加速：如需启用CUDA加速，请确保已安装带CUDA的PyTorch版本
* 内存建议：处理大图像（>1024×1024）建议16GB以上内存

使用说明

图像修复模块

1. 选择修复模式（划痕/污渍）
2. 上传图像文件（支持jpg、png、jpeg格式）
3. 根据模式进行交互（手动绘制划痕/自动生成污渍）
4. 调整修复参数（Patch大小、字典大小、稀疏度等）
5. 执行修复并查看质量评估报告

音频降噪模块

1. 选择音频来源（默认测试信号/上传WAV文件）
2. 对于上传文件，选择处理模式
3. 系统自动添加可控噪声并执行降噪
4. 查看波形对比和SNR提升效果

图像去噪评估

1. 上传测试图像
2. 选择噪声类型（高斯噪声/椒盐噪声/像素丢失）
3. 调整噪声强度和算法参数
4. 查看PSNR/SSIM指标和图像对比

注意事项

* 确保 scratch.py 和 stain.py 与 app.py 在同一目录下
* 音频处理建议使用10秒以内的文件以加快处理速度
* 图像修复模块支持GPU加速，会自动检测CUDA可用性
* 椒盐噪声和像素丢失处理仅为对比参考，核心图像去噪算法针对高斯噪声
